# Workshop Starter

Готовый статический сайт, уже подключённый к общей базе данных
(Supabase) — бэкенд-сервер не нужен, всё общение с базой идёт прямо из
браузера.

## Как начать
Просто опишите свою идею своими словами первым сообщением, например:

> Общий список, где мы с друзьями голосуем, куда пойти поесть.

Claude уже знает, что делать (см. CLAUDE.md) — при необходимости задаст
один короткий уточняющий вопрос, затем перепишет index.html под вашу
идею и покажет, что данные реально сохраняются в базе.

## Как посмотреть готовое приложение
1. В настройках вашего репозитория на GitHub: Settings → Pages → Source →
   выберите ветку main (или ту, куда запушены изменения) и папку root.
2. Через минуту-две сайт появится по адресу вида
   `https://<ваш-логин>.github.io/<название-репозитория>/`
3. Откройте эту ссылку — это ваше работающее приложение.

## Важно
- Все участники используют одну общую базу данных, но каждый видит
  только свои данные (это настроено автоматически через PARTICIPANT_ID
  внутри index.html — трогать не нужно).
- Данные не приватные в строгом смысле — это временная база для
  воркшопа, не для чувствительной информации.
